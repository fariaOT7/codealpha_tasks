# Herbs by Adan – Organic Website UI/UX Design

## About the Project

**Herbs by Adan** is a UI/UX design project created in Figma for an organic products business. The goal of this project was to redesign the website with a clean, modern, and user-friendly interface while making the overall shopping experience more intuitive.

The project includes two Figma files/pages: one for the wireframe and another for the final high-fidelity UI design.
